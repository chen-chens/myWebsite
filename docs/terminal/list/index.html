命令提示字元指令紀錄

指令意思備註
cd 路徑移動到指定路徑。--
cd ../移動到目前路徑的上一層。--
pwd目前路徑位置--
ls條列目前路徑內的所有內容。Mac 指令用法。
dir條列目前路徑內的所有內容。Window 指令用法。
clear清除 目前 Terminal CLI的所有訊息--
rm刪除檔案用 rm 刪檔，連垃圾桶裡都看不到，真的全刪。
mkdir建立資料夾。(make a directory)mkdir 資料夾名稱
touch建立檔案。touch index.html
&&執行完前面指令，接者執行A && B : 執行完 A 後， 執行 B
指定依賴 -v查詢指定依賴目前安裝版本。--
ctrl+C跳出目前執行的環境。顯示提示：(To exit, press Ctrl+C again or Ctrl+D or type .exit)
