Overview of Search Engine Work​

(一) Crawling​

  • Crawling Queue
  • Send Http request By URL
    • 2XX : Parse HTML
    • 3XX : Redirect URL
    • 4XX : Won't parse HTML
    • 5XX : Will come back later --> Crawling Queue
redirect status code
Status CodeDescriptionNoteMore
301Moved permanentlyRedirect new URL.
Allowed to change HTTP method POST to GET when redirecting.		Most common used
308Permanent RedirectRedirect new URL.
Not Allowed change HTTP method when redirecting..		Next.js used
302FoundRedirect URL temporarily.ex.
Redirect to promotion page.

You can trigger a 308 redirect in Next.js by returning a redirect instead of props in the getStaticProps() function.

  • Next 官網
    {
        permanent: true; // trigger 308
    }
client-side error
Status CodeDescriptionNoteMore
404Not FoundClient side error.Can customized 404 page in Next.js.
410GoneClient side error.
Indicates the target URL is no longer available at the server. Won't existed permanently.		ex.
Products permanently removed from inventory.
Blog post removed.
    {
        notFound: true; // trigger 404 & redirect to customized 404 page
    }
server-side error
Status CodeDescriptionNoteMore
500Internal Server Error--Can customized 404 page in Next.js.
503Service UnavailableThe server is not ready to handle the request.
Common causes are a server that is down for maintenance or that is overloaded.		Use 503 for long-term server maintance.
This prevents losing rankings on a long-term basis.
