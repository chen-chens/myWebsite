區塊尺寸計算

block物件的寬度 = block物件 + margin + padding + border。​

從 Google 點擊開發者工具，可以看到區塊尺寸計算方式：

盒模型​

類型盒模型計算方式box-sizing 支持補充
content-box內容物本身Ｏ--
padding-box內容物本身 + paddingＯ--
border-box內容物本身 + padding + borderＯ在計算 RWD 排版時常使用。
margin-box內容物本身 + padding + border + marginＸmargin 為物件與物件之間的距離。

練習: 設定基礎物件 content-box​

style.css
    div{
        width: 300px;
    }
index.html
    <body>
        <div>
            ...
        </div>
    </body>

padding-box​

物件寬度計算 : 內容物本身 + padding = 300px + (25x2)px = 350px​

border-box​

物件寬度計算 : 內容物本身 + padding + border = 300px + (25x2)px + (10x2)px = 370px​

margin-box​

物件寬度計算 : 內容物本身 + padding + border + margin = 300px + (25x2)px + (10x2)px + (10x2)px = 390px​

補充：block 與 inline、inline-block 的差異？​
